# Landing Page Images

This directory contains images for the landing page slideshow.

## Required Images

Please add the following images to this directory:

1. **slide-photography.jpg** - A beautiful photography/wedding scene
2. **slide-catering.jpg** - A catering/venue scene with food
3. **slide-music.jpg** - A music/DJ/entertainment scene
4. **slide-events.jpg** - A celebration/party scene

## Recommended Specifications

- **Resolution**: 1920x1080 pixels (Full HD) or higher
- **Format**: JPG or WebP for best performance
- **Aspect Ratio**: 16:9
- **File Size**: Optimize to under 500KB each for fast loading

## Image Sources

You can get free high-quality images from:
- [Unsplash](https://unsplash.com)
- [Pexels](https://pexels.com)
- [Pixabay](https://pixabay.com)
